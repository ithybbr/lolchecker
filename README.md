# lolchecker
telegram bot that notifies when someone plays league of legends
there are 9 commands
    '/q <name> - to check if someone is playing\n' can be empty
    '/ping <text> - to ping the group with a custom message\n'
    '/clashwhen - to check when the next clash is\n'
    '/poll <text> - to create a poll for the group chat\n'
    '/stop - stops the replied poll'
    '/stats <name> - to check the stats of a player\n' can be empty
    '/matches <name> - to get last 5 games of a player\n'
    '/chat <text> - to prompt gemini\n'
    '/rate <cur1> <cur2>- to get currency exchange rate'
If you want to use it better fork it because league of legends region and default values are hardcoded for my needs.
